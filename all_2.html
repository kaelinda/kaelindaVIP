解构赋值

变量的解构：

  • 解构赋值允许指定默认值。
  • 解构赋值必须保证赋值方 被赋值方类型一直 但Item允许有遗漏或者多出。
  • 默认值可以引用解构赋值的其他变量，但该变量必须已经声明
let [x = 1, y = x] = [];     // x=1; y=1
let [x = y, y = 1] = [];     // ReferenceError
