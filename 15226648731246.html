输入/输出 重定向

命令 说明
command > file 先清空文件中内容，将输出重定向到文件
command < file 将输入重定向到文件
command >> file 将输出以追加的形式重定向到file
N > file 将描述符为N的文件，重定向到文件file
N >> file 将描述符为N的文件，追加到file
n>&m 将输出文件m 和  n合并
n<&m 将输入文件m 和 n合并
<<tag 将开始标记为tag何结尾标记为tag 之间的内容作为输入
